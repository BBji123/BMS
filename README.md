# BMS
基于spring boot+spring security+vue的前后端分离项目: 后台管理系统
# 项目简介
本项目是基于Springboot + Spring Security + Vue的前后端完全分离项目, 实现对用户的简单CURD功能. 
# 运行环境
 Springboot 2.6.2
 Spring Security 5.6.1
 Vue 3.0
 Mysql 8.0
